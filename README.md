# To-Do List Web App

A simple and interactive To-Do List application built with HTML, CSS, and JavaScript. This app allows users to add, mark as completed, and delete tasks, with data persistence using localStorage.

## Features

- **Add Tasks**: Enter a task in the input field and click "Add" to include it in your list.
- **Mark as Completed**: Click on a task to toggle its completion status (strikethrough effect).
- **Delete Tasks**: Click the "×" icon next to a task to remove it from the list.
- **Data Persistence**: Tasks are saved in the browser's localStorage, so they persist across page reloads.

## Technologies Used

- **HTML**: Structure of the web page.
- **CSS**: Styling for a clean and responsive design with a gradient background.
- **JavaScript**: Functionality for adding, completing, and deleting tasks, plus localStorage integration.

## How to Use

1. Clone or download the repository.
2. Open `Index.html` in your web browser.
3. Start adding tasks using the input field and "Add" button.
4. Click on tasks to mark them as done.
5. Click the "×" to delete tasks.

## Project Structure

- `Index.html`: The main HTML file.
- `style.css`: CSS file for styling.
- `script.js`: JavaScript file for functionality.
- `images/`: Folder containing icons or images used in the app.
